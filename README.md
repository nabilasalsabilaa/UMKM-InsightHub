# UMKM-InsightHub
UMKM InsightHub is an AI-based decision support system designed to help Micro, Small, and Medium Enterprises (UMKM) assess their digital readiness and receive strategic recommendations to improve business performance.

## Features
- UMKM segmentation using Machine Learning (Clustering)
- Digital Readiness Score (0–100)
- AI-based recommendation system
- Business performance simulation (impact estimation)
- Interactive dashboard (Streamlit / Power BI)

## Tech Stack
- Python (Pandas, Scikit-learn)
- SQL / BigQuery
- Streamlit / Power BI
- Microsoft Azure (ML & Storage)

## Project Structure
- `data/` → raw & processed dataset
- `notebooks/` → EDA & experimentation
- `src/` → core logic (modeling, scoring, simulation)
- `app/` → Streamlit dashboard
- `models/` → saved models
